Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
div div block wraps a section of code, used for styling purposes
header header block wraps the header of a website, used for styling purposes
ol unordered list bulleted list tag
ul ordered list numbered list

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make bold text <strong>technical sophistication</strong> technical sophistication
a anchor hypertext link to another document or external site <a>technical sophistication</a> Learn Enough
img image inserts an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code insert markup or source code <code>img</code> technical sophistication
span span wrap certain text to be styled later with CSS <span></span> hello there
li list item creates a list item in an ordered or unordered list <li></li>
